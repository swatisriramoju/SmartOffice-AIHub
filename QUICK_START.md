## Troubleshooting

### Server won't start
```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser
```

### Port in use
```powershell
netstat -ano | findstr :8080
taskkill /PID <PID> /F
```

### Page won't load
- Hard refresh: `Ctrl+Shift+F5`
- Clear cache: `Ctrl+Shift+Delete`
- Check console: `F12`
